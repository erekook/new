GEMS PC v.s. GPU
 
We compare the two regular PCs installed with Windows Vista and GEMS software 6.0 with the GPU workstation installed with other commercial FDTD software for a cell phone model simulation. The number of unknowns (6.4Mcells), excitation source type (lumped port), output parameters (far field pattern at 2 frequencies) are same in the simulations of GEMS PC and GPU workstation. Two GEMS PCs, which are directly connected each other by a 3-meter long regular network cable, are much faster than the GPU workstation. The two computers (their total cost is about US$2000.00) can be used as the general purpose when they are not used for the GEMS simulation.
