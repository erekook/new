Troubleshooting
GEMS Installation issues
Check if you install Microsoft .NET 2.0, and try again;
Uninstall the old version and then try again.
Mesh design problems
Use the half finest dimension as the minimum cell size and adjust the ratio to be 1.05 to 1.2;
Ensure that the largest cell size is smaller than 15 cells per wavelength;
Check if you have sufficient memory.
Excitation problems
Waveport: check the direction;
Lumpport: check if the excitation option is checked;
Mode excitation: check if the mode excitation is enabled;
Planewave: check the polarization and incident direction.
Output setting problems
Check the output option is inside the domain;
Check if the output drawing is assigned the output property.
Simulation problems
Check if the convergency criterion is low enough;
Check if the time domain signal is convergent.
Data post-processing problems
You can export the result in the result tree;
If the expected parameters are not in the result tree, use the data post-processing center to generate the desired results.
